### 🌀 Anti-Gravity Mode *(Press G)*

> The most insane feature of the game!

- 🟣 **Toggle with `G` key** at any time during gameplay
- Animals **float upward** and drift in random directions
- Bullets **curve upward** instead of going straight
- **Purple/blue tint** covers the screen with floating star particles
- Glowing **"↑ ANTI-GRAVITY MODE ↑"** banner appears at the top
- Animals look **confused and upside-down**
- Gravity returns to normal when toggled off

---

### 🎮 Gameplay Mechanics

- **Wave System** — Each wave spawns more animals, faster and tougher
- **Lives System** — You have ❤️❤️❤️ 3 lives; animals that escape cost you one
- **Reload System** — Each gun has limited ammo and a real reload time
- **Score Multiplier** — Kill rare animals for higher points
- **Particle Effects** — Bullet impacts, muzzle flashes, death explosions
- **Custom Crosshair** — Yellow precision crosshair follows your mouse
- **Hit Flash** — Screen flashes red when you take damage
- **Sound Effects** — All guns and events have unique Web Audio sounds
- **Animated Background** — Scrolling clouds, twinkling stars, layered scenery

## 🎮 Controls

| Key / Action | Function |
|---|---|
| 🖱️ **Left Click** | Shoot toward cursor |
| `1` | Switch to **Pistol** |
| `2` | Switch to **Shotgun** |
| `3` | Switch to **Machine Gun** |
| `4` | Switch to **Sniper** |
| `R` | **Reload** current gun |
| `G` | Toggle **Anti-Gravity Mode** |
| **Hold Click** | Auto-fire (Machine Gun) |

## ▶ How to Run

No installation. No dependencies. Just one file!

### Option 1 — Direct Open
```bash
# Download or clone the repo
git clone https://github.com/ADARSH685-BOT/-gun-shooting-game-.git

# Go into the folder
cd -gun-shooting-game-

# Open the file in your browser
open shooting-game.html        # macOS
start shooting-game.html       # Windows
xdg-open shooting-game.html    # Linux
```

### Option 2 — Drag & Drop
1. Download `shooting-game.html`
2. Drag it into any browser window (Chrome, Firefox, Edge)
3. 🎮 Start playing instantly!

> ✅ Works 100% offline — no internet required after download

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **HTML5 Canvas API** | All game rendering (animals, bullets, particles, background) |
| **CSS3** | UI styling, animations, anti-gravity effects, crosshair |
| **Vanilla JavaScript** | Full game engine, physics, collision detection, wave system |
| **Web Audio API** | Procedurally generated gun sounds and effects |
| **Google Fonts** | Press Start 2P + Orbitron for arcade typography |

> 🏆 Zero external libraries. Zero frameworks. Pure browser power.

---

## 🧠 How It Works

```
Game Loop (requestAnimationFrame)
    │
    ├── Update Background (stars, clouds, trees)
    ├── Update Animals (movement, arc physics, anti-gravity float)
    ├── Update Bullets (trajectory, anti-gravity curve)
    ├── Check Collisions (bullet ↔ animal hit detection)
    ├── Update Particles (muzzle flash, death bursts)
    ├── Manage Waves (spawn timer, difficulty scaling)
    └── Render Everything → Canvas
```
